# 1、Spring TX
`描述：`
1. 如下所示, 所有的方法都在同一个类中定义(UserService)
2. 外部调用methodB方法时, 如果内部执行method3出现异常那么 methodA方法内部调用后的结果会不会进行回滚？说明原因！

``` java
@Service
public class UserService {

	//insert操作
	public String method1(){
		//do something1 
	}
	//update操作
	public String method2(){
		//do something2
	}
	//delete操作
	public String method3(){
		//do something3
	}

	//添加事务
	@Transactional
	public void methodA(){
		method1();
		method2()
	}
	
	//添加事务
	@Transactional
	public void methodB(){
		methodA();
		method3();	//调用异常
	}
}
```
